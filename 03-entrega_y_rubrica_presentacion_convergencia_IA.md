# Entrega y rúbrica

## Qué tiene que entregar cada grupo

Cada grupo deberá entregar los siguientes materiales:

- Un archivo con la `presentación` que se expondrá en clase.
- Un documento con el `tema elegido` y el `ejemplo real, novedoso o diseñado por el grupo`. Se proporciona una plantilla para facilitar la organización de esta información.
- Un archivo con `preguntas sobre la presentación` para que puedan incorporarse a un cuestionario posterior en Moodle.

## Requisitos de las preguntas

- Deben referirse a contenidos explicados en la presentación.
- Deben ser claras, concretas y estar correctamente redactadas.
- Deben permitir comprobar si los compañeros han comprendido las ideas principales.
- Deben evitar ambigüedades o respuestas dudosas.

## Rúbrica de evaluación

### 1. Comprensión del tema escogido - 3 puntos

- `3 puntos`: El grupo demuestra un dominio claro del tema, lo explica con precisión y utiliza correctamente los conceptos principales.
- `2 puntos`: El grupo comprende el tema de forma adecuada, aunque hay alguna imprecisión menor.
- `1 punto`: La explicación es superficial o presenta varias confusiones importantes.
- `0 puntos`: No se aprecia comprensión suficiente del tema.

### 2. Calidad y relevancia del ejemplo elegido - 2 puntos

- `2 puntos`: El ejemplo está bien elegido, resulta relevante y se relaciona claramente con el tema.
- `1 punto`: El ejemplo guarda relación con el tema, pero está poco desarrollado o no resulta del todo convincente.
- `0 puntos`: El ejemplo no se ajusta al tema o no está suficientemente justificado.

### 3. Calidad de la presentación oral - 2 puntos

- `2 puntos`: La exposición es clara, ordenada, bien estructurada y comprensible para el grupo clase.
- `1 punto`: La exposición se entiende, aunque presenta problemas de organización, claridad o ritmo.
- `0 puntos`: La presentación resulta confusa, desordenada o difícil de seguir.

### 4. Calidad de los materiales entregados - 1,5 puntos

- `1,5 puntos`: Los archivos entregados están completos, bien presentados y recogen correctamente la información solicitada.
- `1 punto`: La entrega es adecuada, pero presenta carencias menores de formato, organización o contenido.
- `0,5 puntos`: La documentación está incompleta o poco cuidada.
- `0 puntos`: No se entrega lo requerido.

### 5. Calidad de las preguntas para Moodle - 1,5 puntos

- `1,5 puntos`: Las preguntas son claras, útiles, bien redactadas y adecuadas para comprobar la comprensión de la presentación.
- `1 punto`: Las preguntas son válidas, aunque mejorables en claridad o relevancia.
- `0,5 puntos`: Las preguntas son poco precisas, demasiado simples o poco útiles.
- `0 puntos`: No se entregan preguntas o no sirven para elaborar el cuestionario.

## Calificación total

- `10 puntos` en total.
